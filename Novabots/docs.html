Bienvenue dans la documentation de Novabot

Cette documentation te guide pour installer, configurer et administrer Novabot sur ton serveur Discord.

Commence par inviter le bot, configure les permissions de base, puis explore les différents modules (modération, logs, automatisations, etc.).

Inviter Novabot sur ton serveur

  1. Clique sur le bouton Ajouter Novabot sur la page d’accueil.
  2. Sélectionne le serveur sur lequel tu veux ajouter le bot.
  3. Vérifie que les permissions cochées correspondent à tes besoins.
  4. Valide l’autorisation et attends que Novabot rejoigne ton serveur.

Une fois le bot ajouté, assure‑toi que son rôle est au dessus des rôles qu’il doit gérer dans l’onglet Rôles de Discord.

Permissions requises

Pour fonctionner correctement, Novabot a besoin de certaines permissions Discord (lecture/écriture, gestion des messages, bannissement, etc.).

  • Gérer les messages : suppression automatique, anti‑spam.
  • Exclure / bannir des membres : commandes de modération.
  • Gérer les rôles : attribution de rôles automatiques.

Tu peux désactiver certaines fonctionnalités si tu ne souhaites pas accorder toutes les permissions.

Configuration de base

Après avoir invité Novabot, commence par définir le préfixe, les salons par défaut et la langue.

Utilise la commande /config ou le dashboard pour modifier ces paramètres à tout moment.

Module modération

Le module de modération permet de gérer les warns, mutes, kicks, bans ainsi que l’anti‑raid et l’anti‑spam.

Tu peux ajuster la sévérité des sanctions, les durées par défaut et les salons où les logs de modération sont envoyés.

Logs & sécurité

Les logs te permettent de suivre ce qui se passe sur ton serveur (sanctions, joins/leaves, modifications de rôles, etc.).

Configure un salon dédié aux logs dans le dashboard ou via la commande prévue pour garder une trace des actions importantes.

Gérer ses licences

Les licences te donnent accès aux fonctionnalités premium de Novabot (performances avancées, modules supplémentaires, support prioritaire).

Connecte‑toi sur le Dashboard avec ton compte Discord pour voir la liste de tes licences, leur statut et leurs dates d’expiration.

Facturation & paiements

Les paiements sont traités via nos prestataires sécurisés (par exemple Stripe ou PayPal, selon ce que tu utiliseras).

En cas de question concernant une facture ou un renouvellement, ouvre un ticket sur le serveur support Novabot avec l’e‑mail de paiement.

FAQ

Novabot est‑il gratuit ?

Une version gratuite est disponible avec les fonctionnalités essentielles. Les licences ajoutent des options avancées pour les serveurs exigeants.

Le bot fonctionne‑t‑il 24/7 ?

Oui, l’infrastructure est pensée pour une disponibilité maximale, mais des maintenances ponctuelles peuvent avoir lieu.

Support

Pour toute question ou problème, rejoins le serveur Discord de support Novabot et ouvre un ticket dans le salon prévu à cet effet.

Merci de fournir un maximum d’informations (ID du serveur, captures, commandes utilisées) afin de nous aider à résoudre ton problème rapidement.
